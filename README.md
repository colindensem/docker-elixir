# elixir
Docker, Debian base, Elixir


## To Build

```
$ docker build -t=colindensem/elixir -rm=true .
```

## To Run

```
$ docker run -i -t colindensem/elixir /bin/bash
```


## Next Steps
This is intended to use this image as a base for other projects to build upon.
